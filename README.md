# Udemy Dark Mode

Udemyの全ページにダークモードを適用する軽量なChrome拡張機能です。  
「正解」「不正解」などの重要なフィードバックも視認性を高めた配色で強調します。

尚、for文でぶん回してるので結構適当にやってます。ご了承ください。。。

## 🚀 主な機能

- Udemyの全ページを自動でダークテーマ化
- SPA（Single Page Application）対応：動的に変わるページでも適用を維持
- 「正解」は緑、「不正解」は赤で強調表示
- 拡張機能のポップアップからダークモードのON/OFF切り替えが可能
- 設定は `chrome.storage.local` に保存

## 🛠 インストール手順（開発用）

1. このリポジトリをクローンまたはZIPでダウンロード
2. Chromeで `chrome://extensions/` を開く
3. 右上の「デベロッパーモード」を有効にする
4. 「パッケージ化されていない拡張機能を読み込む」からこのフォルダを選択
5. Udemyのサイトを開いて動作確認
